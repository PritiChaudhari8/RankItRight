# RankItRight

**RankItRight** is a Resume and Video Evaluation system designed for HR professionals and students. It leverages **Natural Language Processing (NLP)**, **Machine Learning (ML)**, and **Generative AI (Gen AI)** to rank resumes and analyze soft skills effectively.

---

## Features

- **Resume Ranking:** Automatically score and rank resumes based on keywords, experience, and skills.
- **Soft Skills Analysis:** Evaluate communication and other soft skills from uploaded content or videos.
- **Machine Learning:** ML models used to classify, rank, and score resumes.
- **Gen AI:** AI-powered suggestions and evaluation for better decision-making.

---

## Technologies Used

- **Languages:** Python, MySQL, HTML, CSS
- **Libraries & Tools:** NLP libraries (like NLTK/spaCy), ML frameworks (like scikit-learn, TensorFlow), Gen AI APIs
- **Database:** MySQL

---

## How it Works

1. Upload resumes and video content.
2. System analyzes resumes and scores them using ML/NLP models.
3. Soft skills are assessed using Gen AI techniques.
4. Outputs ranked list of candidates and skill analysis reports.

---

## Future Enhancements
